# Getting Started
To start the application, navigate to the project directory and run:
```bash
npm install
```
This will install all the required dependencies.

Next, run:

This will start the Angular development server, and you can access your application at http://localhost:3000.

Running Server.js

To start the server.js file, navigate to the project directory and run:

```bash
npx serve
```
This will start the server, and you can access it at http://localhost:3000.
